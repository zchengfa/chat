# chat

#### 介绍
本项目是一个聊天项目，目前项目实现了私聊跟群聊，更换头像功能，支持发送图片、文字以及表情

#### 安装

1. 克隆项目
2. 执行npm install命令安装项目所需依赖包

#### 使用
执行 npm run start命令即可

#### 运行在桌面端
运行前请确保是否已安装了electron,若已安装即可执行 npm run electron指令
